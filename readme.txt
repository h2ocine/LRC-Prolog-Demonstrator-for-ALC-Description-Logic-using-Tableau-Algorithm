Développement d'un démonstrateur Prolog basé sur l'algorithme des tableaux pour la logique de description ALC.

Ce projet implémentant un démonstrateur Prolog utilisant l'algorithme des tableaux pour vérifier la syntaxe et la sémantique de la logique de description ALC, ainsi que pour démontrer l'insatisfiabilité des assertions.
